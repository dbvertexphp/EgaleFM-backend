<h2>⚙️ Installation</h2>

### 📦 Install Dependencies

```bash
npm install
```

### 🔧 Setup Environment

```bash
copy .env.example .env
```

Update `.env`:

```
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
PORT=5000
```

---

### ▶️ Start Server

```bash
npm start
```

### 🔄 Development Mode

```bash
npm run server
```
